Readme Spanish Add-On

This add-on is for use with the Spanish translation of BP-BGT Worldmap. Using it gives you a worldmap with writing in Spanish.

You apply the add-on by placing the included folder "moz" into the folder bp-bgt-worldmap/language/spanish of the BP-BGT Worldmap mod.
On Windows you can do this simply by extracting the add-on on top of the extracted contents of the main BP-BGT Worldmap package.

This version of the addon is only compatible with the main BP-BGT Worldmap Mod of v3.10 or higher.
